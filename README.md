
# Superstore Sales Data Analysis

## Objective
Analyze sales data to identify trends and improve business decisions.

## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Key Tasks
- Data cleaning
- Feature engineering
- Exploratory Data Analysis
- Visualization

## Insights
- Technology category has highest sales
- Discounts reduce profitability

  dataset link : https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting
